# ✨ **Features**

React Image Editor is a comprehensive AI-powered media editing platform with advanced features for image processing, AI integration, and collaborative workflows.

---

## 🎨 **Core Image Processing**

### **Basic Operations**
- **Crop & Resize**
  - Smart crop with aspect ratio preservation
  - Multiple resize algorithms (Lanczos, Bicubic, Nearest)
  - Batch resize operations
  - Custom dimension input

- **Filters & Effects**
  - Blur (Gaussian, Motion, Radial)
  - Sharpen (Unsharp Mask, High Pass)
  - Brightness & Contrast adjustment
  - Vintage & Retro filters
  - Color correction and grading
  - Noise reduction

- **Transformations**
  - Rotate (90°, 180°, 270°, custom)
  - Flip (Horizontal, Vertical)
  - Perspective correction
  - Skew correction
  - Warp transformations

- **Format Support**
  - JPEG, PNG, WebP, GIF, BMP
  - TIFF, SVG (import)
  - RAW format support (premium)
  - HEIC support (mobile)
